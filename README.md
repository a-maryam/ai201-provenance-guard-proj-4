# Provenance Guard

## Detection Signals
1. **LLM-based classification:** semantic assessment by LLM. It will catch if the meaning or style together read like AI or human. Output is a score 0 - 1 
- The LLM will miss structure which is really uniform. 
2. **Stylometric heuristics:** Structural checks in Python. Checks statistical properties of the text to see if consistent with human or AI, such as, punctuation, sentence length, etc. Output is a score 0-1
- The stylometry has no concept of meaning, so something that is not structurally off can be AI-generated in content.

## Confidence Scoring
- scoring formula: confidence = (llm_score * 0.65) + (style_score * 0.35)
- Testing: I tried AI-generated examples, I tried some examples that I typed up on my own, and used examples from the project instructions to see that the scoring was mostly in the correct direction either way: human or AI.

## Known Limitations
- Short texts are difficult to label as AI because there is not enough stylometry data. 
Clearly AI-generated example:  Artificial intelligence represents a transformative paradigm shift in modern society. It is important to note that while the benefits of AI are numerous, it is equally essential to consider the ethical implications. Furthermore, stakeholders across various sectors must collaborate to ensure responsible deployment.
- This only scored 0.65 due to its length.

## Rate Limiting
- /submit is rate limited to 10 requests per minute and 100 requests per day per IP address to prevent spam and abuse. This is the default but it is a good default because 10 requests per minute is a lot of this tool and it doesn't really make sense for users to make 100 requests per day either.

## Spec Reflection
- one way the spec guided me: I made sure to avoid false positives. So the threshold for high-confidence AI is high at >=.80.
- one way I diverged: The spec defined the confidence formula as `(llm_score × 0.6) + (style_score × 0.4)`. I shifted the weights to `(llm_score × 0.65) + (style_score × 0.35)` because the LLM semantic signal proved more discriminative than the stylometric heuristics, especially on short texts where structural features are sparse.
